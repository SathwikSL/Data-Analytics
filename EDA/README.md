# 📊 Exploratory Data Analysis (EDA) Projects using Python

This repository contains two Exploratory Data Analysis (EDA) projects built using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**. The projects focus on cleaning, analyzing, and visualizing datasets to extract meaningful insights.

---

## 📁 Repository Structure

```
EDA-Projects/
│
├── store_sales.csv          # Store Sales Dataset
├── menu.csv                 # Food Menu Dataset
│
├── store_sales_analysis.py  # Store Sales EDA
├── menu_analysis.py         # Menu Dataset EDA
│
└── README.md
```

---

# 📌 Project 1: Store Sales Analysis

## 📖 Description

This project performs Exploratory Data Analysis (EDA) on a retail store sales dataset. The analysis helps understand customer purchasing behavior, product performance, seasonal trends, payment methods, discounts, and customer ratings.

## 📊 Analysis Performed

- Data Loading
- Data Inspection
- Missing Value Check
- Duplicate Removal
- Statistical Summary
  - Mean
  - Median
  - Mode
  - Standard Deviation
- Gender Distribution
- Age Distribution
- Product Category Analysis
- Revenue by Category
- Top 10 Purchased Items
- Seasonal Sales Analysis
- Payment Method Analysis
- Discount vs Amount Analysis
- Previous Purchases Analysis
- Item Rating Analysis
- Correlation Heatmap

## 📈 Visualizations

- Count Plot
- Histogram
- Horizontal Bar Chart
- Bar Chart
- Scatter Plot
- Correlation Heatmap
